# 내연봉계산기 - Swift/SwiftUI/iOS

내 연봉계산기는 연본 실수령액 및 기타 공제금액을 확인 할 수 있는 아이폰용앱입니다.
연봉목록을 보거나 연봉을 입력하여 실수령액, 세금, 보험금 등을 확인 할 수 있습니다.

2021년 소득세, 국민연금, 건강보험 기준 데이터가 포함되어 있습니다.
